# projects

A selection of things I've built over the years, from open source
infrastructure used by millions to small experiments built for fun.

## major projects

- [**Model Context Protocol**](https://modelcontextprotocol.io) — I am the
  co-creator of MCP, an open standard for connecting AI assistants to the
  tools, data and systems they work with.
- [**Mercurial**](https://mercurial-scm.org) — a core contributor to the
  distributed version control system. I developed key features including
  bookmarks, the pager, and shelving.
- [**Git**](https://git-scm.com) — contributor to the version control system.
- [**PHP**](https://php.net) — a core developer of the language and the
  release manager responsible for the official releases of PHP 5.4 and PHP 5.5.

## fun projects

- [**Pattern**](https://pattern.experimentalworks.net) — a 320x320px grid that
  anyone can write to by picking their favorite color. A small, meaningless,
  fun experiment.
- [**neweden**](https://github.com/dsp/neweden) — a Rust library for wayfinding
  in EVE Online.
- [**yubikey-kube-proxy**](https://github.com/dsp/yubikey-kube-proxy) — a local
  proxy for Kubernetes authentication using a YubiKey PIV.
- [**mled**](https://github.com/dsp/mled) — translates stdin to Morse code and
  blinks your Caps Lock LED accordingly.
- [**v6tools**](https://github.com/dsp/v6tools) — a set of PHP tools for working
  with IPv6.

More on [GitHub](https://github.com/dsp).
